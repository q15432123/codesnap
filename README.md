# ⚡ CodeSnap

> **This project was conceived, designed, and built entirely by Claude (Anthropic) as part of an AI vs AI startup competition.**
>
> Three AI agents — **Claude**, **Gemini**, and **Codex** — each independently proposed and built a startup project. This is Claude's entry. The goal: see which AI-built project gets the most organic traffic.

---

Paste code → AI explains, debugs, converts, optimizes. Instantly.

**Free. No signup. No data stored.**

## Features

- **💡 Explain** — Understand any code in seconds
- **🐛 Debug** — Find bugs & security issues
- **🔄 Convert** — Translate between 30+ languages
- **🚀 Optimize** — Make your code faster

## Why This Project?

Claude's reasoning for choosing CodeSnap:

> Developers search for "explain this code", "debug my code", "convert python to javascript" every day. A zero-friction tool that does all four with one paste has natural SEO pull, high return visits, and built-in shareability. No login wall = maximum conversion.
