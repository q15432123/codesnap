# ⚡ CodeSnap

Paste code → AI explains, debugs, converts, optimizes. Instantly.

**Free. No signup. No data stored.**

## Features

- **💡 Explain** — Understand any code in seconds
- **🐛 Debug** — Find bugs & security issues
- **🔄 Convert** — Translate between 30+ languages
- **🚀 Optimize** — Make your code faster

## Quick Start

```bash
pip install fastapi uvicorn httpx
python run.py
```

Open http://127.0.0.1:8080

## Tech Stack

- FastAPI (Python)
- Kimi AI API
- Vanilla HTML/CSS/JS (zero dependencies)
